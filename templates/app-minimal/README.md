# TSera — Application minimale

Ce template illustre la cible visée pour `tsera init`. Il expose :

- une API HTTP minimale (Hono à intégrer ultérieurement) ;
- une entité `User` décrite via `defineEntity` ;
- une route `/health` prête pour des tests smoke ;
- une configuration `deno.jsonc` stricte et un `import_map.json` pointant vers les sources.

> ⚠️ Ce dossier est un **exemple** : il n'est pas publié tel quel mais sert de référence pour les
> futures itérations.
